# swobby

Trade books with people near you. A local book swap platform built with React Router 7, Supabase, and Tailwind CSS.

## Setup

Install dependencies:

```bash
npm install
```

Create a `.env` file with the following:

```
VITE_SUPABASE_URL=
VITE_SUPABASE_KEY=
VITE_GOOGLE_MAPS_API=
GOOGLE_BOOKS_API_KEY=
```

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
npm start
```
